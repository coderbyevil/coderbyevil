## <img src="https://media.giphy.com/media/WUlplcMpOCEmTGBtBW/giphy.gif" width="28"> &nbsp;Quick API Demo

```bash
# Generate a disposable email
curl -X POST https://evilmail.pro/api/temp-email \
  -H "X-API-Key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domain": "evilmail.pro", "ttlMinutes": 60}'

# Response
{
  "status": "success",
  "data": {
    "email": "shadow.fox.4821@evilmail.pro",
    "sessionToken": "abc123...",
    "ttlMinutes": 60
  }
}
```
